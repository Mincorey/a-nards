# engine/ — архивный прототип (не используется кодом)

Это исходный прототип движка коротких нард (`nardy-engine.js`) с заметками
(`LOGIC_README.md`, `architecture_map.txt`), с которого начинался проект.

Рабочий движок живёт в `src/engine/` (TypeScript, короткие + длинные нарды,
покрыт тестами) и синхронизируется на сервер в `supabase/functions/_shared/`.
Эти файлы оставлены только для истории (аудит L1). Кодом/сборкой не используются.
